# IBM SRE Academy Final Project

#### Step by step walkthrough on setting up, running and monitoring Minecraft in Kubernetes


## Tools Utilized
|Tools              |Function                                |
|-------------------|----------------------------------------|
|Python 3           |Programming                             |
|pip/venv           |Manages Python packages                 |
|WSL2               |Runs Ubuntu OS on Windows OS            |
|Docker             |Creates and manages containter images   |
|Kubernetes         |Builds and runs containers              |    
|Minikube           |Runs a local Kubernetes cluster         |
|kubectl            |Manages Kubernetes resources            |
|Prometheus         |Monitoring system & time series database|
|Grafana            |Visualization, analytics & monitoring   |
|pipx + Ansible     |Automatization                          |
|Helm               |Manages Kubernetes applications         |
|Minecraft Launcher |Runs Minecraft                          |
