# Open Docker Desktop

- ### In Visual Studio Code

Run ```docker compose up -d```
##
- ### Open Grafana 

Go to  ```http://localhost:3000/``` in web browser.

Login: ```admin/admin```
##
- ### Grafana Dashboard
1. Menu > Connections > Data Sources > Prometheus

2. Connection > Prometheus server URL: ```http://prometheus:9090```

3. Import available dashboards
